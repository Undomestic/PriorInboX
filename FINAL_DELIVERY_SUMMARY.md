## 🎓 How It Works

### Scoring Algorithm
```
Email Input
  ↓
Analyze 7 categories:
  - Urgent keywords → 30-40 pts
  - Deadline keywords → 20-25 pts
  - Opportunity keywords → 25-40 pts
  - Action keywords → 15-30 pts
  - Meeting keywords → 12-20 pts
  - Financial keywords → 15-20 pts
  - Security keywords → 25-40 pts
  ↓
Add sender bonus: +15 for important senders
  ↓
Calculate total score
  ↓
Score ≥ 30? → Mark as Important
```

### User Workflow
```
User clicks "Analyze Emails"
  ↓
System analyzes 100 recent emails
  ↓
Auto-marks emails with score ≥ 30
  ↓
Shows results (analyzed, marked, rate)
  ↓
User views insights & suggestions
```
